My portfolio

Ambitieproject

For this project I was told to really challange myself, and so I did. I made a 2d platformer with the goal of making an enemy that could move and attack. I used al little tutorials as posible and just tried to use everything I learned from previous projects to code the AI.

The hardest part of this project was making the enemy stop and attack the player. when making earlier games I used a ground check to see if the player was touching the ground. for my enemy AI i reused this function to look if the player was nearby, and if he was, the enemy would stop and attack.

Shineproject

this project was meant to be a way to show how mush you've learn using Unity. I really pushed my limit by making a lot of machanics that i had never implemented in one of my games before. I made enemys that would follow you and would elimanate you if you got too close.

I used the players position, and made the enemy's alwasy move in the players direction. the player could fight back by shooting at the enemys. elimanating them in the process. eventualy there would be too many enemys, and the player would be overwelmed.
